## 📌 Deskripsi
Project ini melakukan proses ETL dan visualisasi data recall kendaraan menggunakan Python (Google Colab) dan Tableau.

## ⚙️ Tools
- Python (Pandas)
- Google Colab
- Tableau

## 🔄 Proses
- Extract data dari CSV/Excel
- Data cleaning & transformasi
- Aggregasi data
- Visualisasi menggunakan Tableau
